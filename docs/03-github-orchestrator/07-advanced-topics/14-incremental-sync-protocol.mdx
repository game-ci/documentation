# Incremental Sync

Incremental sync is now documented as part of the
[Standalone Streaming Hot Runner](hot-runner-protocol) model.

The protocol still exists as the workspace update layer, but it is easier to understand alongside
the warm runner process, runner transport, and output stream that use it. See
[Standalone Streaming Hot Runner](hot-runner-protocol#incremental-sync) for the current API,
strategies, and inputs.
